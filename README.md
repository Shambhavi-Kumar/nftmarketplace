# NFT Marketplace

A decentralized NFT marketplace built using Solidity and React.

## Tech Stack
- Solidity
- Hardhat
- React / Next.js
- Ethers.js
- IPFS

## Features
- Mint NFTs
- List NFTs for sale
- Buy NFTs using ETH
- Wallet integration (MetaMask)

## Setup
```bash
npm install
npm run dev
